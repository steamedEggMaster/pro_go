1. 패키지 이름 충돌 해결
	// 같은 패키지 이름이 존재할때
	// 1. 별칭 이용하기
	//    store2 "packages/store"
	// 2. 점(.) import 사용
	//    . "packages/store"
	//    => 접두사 없이 해당 패키지 내부 사용 가능함

2. init 초기화 함수
    1. 해당 패키지에서 자동으로 호출됨
    2. 직접 호출 불가
    3. 단일 파일에서 초기화 함수 여러번 정의 가능
    4. 다른 초기화 함수에 의존적이면 안됨
